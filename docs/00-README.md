## Ключевые принципы

- **Скорость > функциональность.** Оверлей должен открываться <100ms.
- **Стриминг критичен.** Без него UX ломается.
- **Одно окно для MVP.** Два окна — для V0.2+.
- **DeepSeek Chat как основная модель.** GPT-4o-mini как fallback.
- **Electron + React + Tailwind.** Проверенный стек для десктопных UI.
