# 01 — Концепция продукта

## Краткая формулировка

**NovaSync** — это always-available персональный AI-ассистент для Windows/macOS/Linux. Вызывается глобальным хоткеем, принимает запрос через минималистичный оверлей в центре экрана и показывает ответ в компактном floating-виджете в правом нижнем углу.

**Позиционирование:** не чат-бот, не замена IDE, а **контекстный помощник**, который всегда под рукой. Как Spotlight, но с AI. Как Perplexity, но на десктопе. Как Copilot, но для всего, а не только для кода.

---

## MVP-функции (must-have, v0.1)

| # | Функция | Сложность | Приоритет |
|---|---------|-----------|-----------|
| 1 | **Глобальный хоткей** (Alt+Space / Ctrl+Shift+A) — вызов оверлея из любого приложения | Low | P0 |
| 2 | **Центральный оверлей** — тёмное полупрозрачное окно со строкой ввода, появляется мгновенно | Low | P0 |
| 3 | **Отправка текстового запроса** к LLM API, стриминг ответа | Low | P0 |
| 4 | **Floating response widget** — компактный блок в правом нижнем углу с ответом | Medium | P0 |
| 5 | **Выбор модели/режима** — быстрое переключение между моделями из оверлея | Low | P1 |
| 6 | **Копирование ответа** — кнопка «Copy» в виджете | Low | P1 |
| 7 | **История запросов** — локальное хранение последних 50 запросов | Low | P1 |
| 8 | **Системный трей** — иконка в трее, приложение работает в фоне | Low | P0 |
| 9 | **Базовые быстрые действия** — «Объясни», «Переведи», «Summarize» | Medium | P1 |
| 10 | **Автозапуск** — приложение стартует вместе с системой (опционально) | Low | P2 |

## Что НЕ входит в MVP (сознательно отложено)

- Анализ скриншотов экрана
- Голосовой ввод
- System tool calling (запуск программ, работа с файловой системой)
- Плагины и расширения
- Локальные модели
- Мультимодальность (изображения)

> Принцип MVP: сделать **одну вещь идеально** (быстрый вызов → запрос → стриминг-ответ), а не 20 вещей средне.

## Идеи для следующих версий

### V0.2 — Контекст и зрение

| Функция | Сложность | Риск |
|---------|-----------|------|
| **Скриншот экрана** — ручной захват (кнопка в оверлее), отправка в multimodal-модель | Medium | Low |
| **Контекст активного окна** — определение названия приложения и заголовка окна | Low | Low |
| **Буфер обмена** — кнопка «Вставить из буфера» или авто-вставка при вызове | Low | Low |
| **Выделенный текст** — получение через Ctrl+C → clipboard read | Low | Low |
| **Markdown-рендеринг** — поддержка кода, списков, таблиц в ответе | Low | Low |

### V0.3 — Системные действия (уровни доступа)

| Уровень | Функции | Активация |
|---------|---------|-----------|
| **Read-only** | Чтение буфера обмена, имя активного окна, скриншот (ручной), список процессов | По умолчанию |
| **Подтверждаемые** | Создание/редактирование файла в ~/NovaSync/workspace, открытие URL, запись в буфер | Пользователь подтверждает |
| **Опасные** | Произвольный shell, удаление файлов, произвольные сетевые запросы | Отключены, ручное включение |

### V0.4+ — Горизонт

| Функция | Сложность |
|---------|-----------|
| **Голосовой ввод** (Whisper API или Web Speech API) | Medium |
| **TTS-ответ** (ElevenLabs / OpenAI TTS) | Low |
| **Кастомные промпты/пресеты** — «Code Review», «Email Reply» | Low |
| **Локальная модель** — ollama + llama3.2 для офлайн-режима | High |
| **Плагины** — пользовательские расширения (как Raycast extensions) | High |
| **RAG по документам** — индексация локальных файлов | High |
| **Agent mode** — multi-step reasoning с tool calling | High |
| **Continuous screen awareness** — проактивные подсказки | High |
| **Интеграции** — VS Code, Terminal, браузер | High |
